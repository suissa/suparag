Documento de Teste para RAG

Este é um documento de teste para validar o sistema de busca semântica.

Informações importantes:
- O sistema usa embeddings para busca vetorial
- A tabela rag_documents armazena os documentos
- A função match_documents faz a busca por similaridade
- O threshold padrão é 0.5

Exemplo de uso:
1. Fazer upload do documento
2. O sistema gera o embedding automaticamente
3. Fazer perguntas no chat
4. O sistema busca documentos similares
5. Retorna os resultados mais relevantes

Palavras-chave: RAG, embeddings, busca semântica, Supabase, PostgreSQL, pgvector
